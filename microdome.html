Microdome Classes

A Full-Stack Learning Management Platform

Project Overview

Microdome is a full-stack LMS designed for managing courses, users, quizzes, and payments with role-based access for admins, instructors, and students.

My Responsibilities

  • Authentication & authorization (JWT-based)
  • Course, quiz, and user management
  • Payment gateway (Razorpay) integration
  • REST API integration and database design

Tech Stack

React Node.js Express MongoDB JWT
